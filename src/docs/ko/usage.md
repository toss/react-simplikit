# 설치 및 사용하기

Node.js나 Bun을 사용하는 경우, [npm](https://npmjs.com/package/reactie)에서 설치할 수 있어요.

## Node.js

reactie는 Node.js 18 또는 이후 버전을 지원해요. reactie을 설치하기 위해서는 아래 명령어를 사용해주세요.

::: code-group

```sh [npm]
npm install reactie
```

```sh [pnpm]
pnpm add reactie
```

```sh [yarn]
yarn add reactie
```

:::
