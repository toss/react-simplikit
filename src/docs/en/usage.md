# Installation & Usage

reactie is available via [npm](https://npmjs.com/package/reactie) for Node.js and Bun.

## Node.js

reactie supports Node.js 18 or later. Install reactie with the following command:

::: code-group

```sh [npm]
npm install reactie
```

```sh [pnpm]
pnpm add reactie
```

```sh [yarn]
yarn add reactie
```

:::
